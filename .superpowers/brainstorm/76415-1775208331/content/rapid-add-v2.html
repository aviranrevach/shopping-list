Two modes

🔍 Search bar Filters/highlights items already on the current list
+ Button Opens a full-screen rapid-add canvas

Rapid-add canvas flow

  1. Full screen with "הוספת פריטים" header and "סיום" (Done) button
  2. Blank canvas — type an item name, hit Enter
  3. Item appears above with a ✓ mark, cursor moves to next line
  4. Suggestions as chips above the keyboard — tap one to auto-complete, or ignore and keep typing
  5. Each Enter adds and moves to the next line — like writing in Notes
  6. Tap "סיום" when done → back to list with all new items added

Suggestion chips

  • Horizontal scrollable row sitting just above the keyboard
  • Sourced from family item history, matched by what you're typing
  • Tap a chip → auto-fills the line with that item + its category, moves to next line
  • Never blocks your typing — suggestions are passive, not a dropdown
