The flow

  1. Tap + or search bar → keyboard opens, cursor in input
  2. Type → suggestions appear from family history (with category tags)
  3. Tap a suggestion → item added with its remembered category, input clears, stays focused
  4. Or just hit Enter → typed text added as new item (category: "Other"), input clears, stays focused
  5. Keep typing → next item, repeat. Like a notes app.
  6. Tap away or ‹ back → exit rapid-add mode

Key details

  • Subtle confirmation toast after each add (auto-dismisses)
  • Input never loses focus between adds — keyboard stays up
  • New items default to "Other" category — recategorize later via item detail if needed
  • Suggestions come from all family lists, sorted by frequency
